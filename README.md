# Position Size Calculator — AMarkets MT4

Калькулятор розміру позиції для торгівлі через AMarkets MT4. Автоматично розраховує оптимальний об'єм лоту на основі ризик-менеджменту.

**▶ [Відкрити калькулятор](https://1artifintel-art.github.io/micro-futures-tools/calculator.html)**

---

## Як працює калькулятор

### Логіка розрахунку

Основний принцип — фіксований максимальний збиток на угоду в доларах. Калькулятор визначає скільки лотів можна відкрити щоб при спрацюванні стоп-лосу збиток не перевищив задану суму.

```
Збиток на 1 лот = кількість пунктів SL × вартість 1 пункту
Об'єм (лоти)   = Макс. збиток / Збиток на 1 лот
```

### Вхідні параметри

| Параметр | Опис |
|---|---|
| **Символ** | Торговий інструмент |
| **Макс. збиток ($)** | Максимально допустимий збиток на угоду в USD |
| **Стоп-лос (пунктів)** | Відстань від ціни входу до стоп-лосу в пунктах |
| **Ціна входу** | Ціна відкриття позиції (Buy Limit / Sell Limit) |
| **Тейк-профіт** | Цільова ціна для закриття позиції |
| **Напрямок** | Buy (Long) або Sell (Short) |
| **Баланс рахунку ($)** | Поточний баланс для розрахунку % ризику |

### Результати

| Показник | Формула | Опис |
|---|---|---|
| **Об'єм (лоти)** | `Макс.збиток / (SL пунктів × ціна пункту)` | Дробове число, наприклад 0.67 або 2.50 |
| **Макс. збиток** | `Об'єм × SL пунктів × ціна пункту` | Реальний збиток в USD |
| **Потенц. профіт** | `Об'єм × TP пунктів × ціна пункту` | Потенційний прибуток в USD |
| **R:R** | `Профіт / Збиток` | Співвідношення ризик/прибуток |
| **Ціна стоп-лосу** | Buy: `Entry − SL×pipSize` / Sell: `Entry + SL×pipSize` | Точна ціна для виставлення SL |
| **Ризик від балансу** | `Збиток / Баланс × 100%` | Рекомендується не більше 2% |

### Попередження

- 🟡 **Ризик > 2%** — перевищує рекомендований ліміт
- 🔴 **Ризик > 5%** — небезпечно для рахунку

## Приклади розрахунків

### EURUSD — Buy Limit
```
Вхід:        1.13500
Стоп-лос:    1.13000  (50 піпів = 500 пунктів)
Тейк-профіт: 1.15700  (220 піпів)
Макс. збиток: $500

Збиток на 1 лот = 500 × $10 = $500
Об'єм = $500 / $500 = 1.00 лот
Потенц. профіт = 1.00 × 2200 × $10 = $22,000... 

Виправлення: 50 піпів = 500 пунктів, але ціна пункту $10 для 1 пункту (0.00001)
Збиток на 1 лот = 500 пунктів × $10 = ... 

Правильно: 50 піпів × $10/піп = $500 → 1 лот
Профіт: 220 піпів × $10 × 1 лот = $2,200 | R:R = 4.4
```

### XAUUSD — Buy Limit

```
Вхід:        4219.52
Стоп-лос:    4169.52  (SL = 5000 пунктів = $50.00)
Тейк-профіт: 4276.09

Збиток на 1 лот = 5000 × $0.10 = $500
Об'єм = $500 / $500 = 1.00 лот

```

### WTI — Sell Limit

## Ризик-менеджмент

| Ризик на угоду | Оцінка |
|---|---|
| ≤ 1% | ✅ Консервативно |
| 1–2% | ✅ Рекомендовано |
| 2–5% | ⚠️ Підвищений ризик |
| > 5% | 🔴 Небезпечно |

**Рекомендація:** при балансі $5,000 максимальний ризик на угоду = $100 (2%).
