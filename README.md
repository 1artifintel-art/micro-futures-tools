# Micro Futures Tools

Trading tools for AMarkets MT4.

## Position Size Calculator

**[▶ Open Calculator](https://1artifintel-art.github.io/micro-futures-tools/calculator.html)**

Calculates lot size, stop loss price, potential profit and R:R ratio.

Supported instruments: EURUSD, GBPUSD, AUDUSD, NZDUSD, USDCAD, USDCHF, USDJPY, XAUUSD, XAGUSD, WTI, NGAS, S&P500.
